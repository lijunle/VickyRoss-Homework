Including Responsive Forms

Forms are another type of content that require a little extra special attention when using a responsive design. It is important to make sure that people who visit your site on their mobile phones and easily and quickly complete your forms. This can be accomplished using some custom CSS with media queries and, for the most part, avoiding designing forms using tables.

We offer the example responsive form below as a template which can be adapted for use with your existing forms. It is compatible with most any form processing script including FormMail.pl and even comes with a sample basic PHP form script.
